## 如何启用

1. 将这个仓库命名为你的 GitHub 用户名，例如用户名是 `octocat`，仓库名也必须是 `octocat`。
2. 进入仓库 `Settings` -> `Actions` -> `General`，将 `Workflow permissions` 设置为 `Read and write permissions`。
3. 推送后 GitHub Actions 会每 6 小时自动更新，也可以在 Actions 页面手动运行 `Update Profile README`。
4. 如需改默认展示数量，编辑 [scripts/update-profile.mjs](./scripts/update-profile.mjs) 里的 `MAX_REPOS`。
